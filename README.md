# ☕ CHO-CHA-DA (ชอ-ชา-ฎา) ผู้ช่วย SME
> **เพื่อนคู่คิดร้านค้าขนาดย่อม ระบบจัดการหน้าร้านและคลังสินค้าที่ง่ายที่สุด เพื่อพ่อค้าแม่ค้ายุคดิจิทัล**

### 🌟 เริ่มต้นฟรีวันนี้ (Local MVP): ออกแบบมาเพื่อ "ร้านค้าข้างทางและชาวตลาดนัด" อย่างแท้จริง
ไม่ว่าคุณจะเป็นร้านขายของกินบนแผงลอย Street Food, ร้านเสื้อผ้าแฟชั่นในตลาดนัด, บาริสต้าที่กำลังยุ่งในร้านกาแฟสด หรือเจ้าของร้านเบเกอรี่โฮมเมด — **CHO-CHA-DA (ชอ-ชา-ฎา) ผู้ช่วย SME** คือคำตอบที่จะเปลี่ยนหน้าจอโทรศัพท์มือถือของคุณให้กลายเป็นเครื่องคิดเงินอัจฉริยะได้ใน 1 วินาที!

เราเชื่อว่า "ความเร็วคือหัวใจของการค้าขาย" ระบบนี้จึงถูกดีไซน์ภายใต้แนวคิด **"Zero-Learning Curve"** ไม่ต้องเรียนรู้เยอะ ปุ่มใหญ่ชัดเจน สแกนไว ไม่ต้องเสียเวลาพิมพ์ ช่วยให้คุณปิดยอดขายและทอนเงินได้ถูกต้องแม่นยำภายใน 5 วินาที แม้ในชั่วโมงเร่งด่วนที่ลูกค้าต่อคิวแน่นหน้าร้าน โดยทำงานแบบออฟไลน์ได้ 100% ข้อมูลปลอดภัยในเครื่อง ไม่ต้องง้อสัญญาณอินเทอร์เน็ต!

#### 📸 ฟีเจอร์หลักในเวอร์ชันใช้งานฟรี:
*   **สแกนปุ๊บ เข้าตะกร้าปั๊บ (Scan to Cart):** เปิดกล้องยิง QR สินค้าเพื่อรวมยอดขายหลายชิ้นพร้อมกันได้ทันที มีระบบล็อกป้องกันการสแกนซ้ำซ้อนโดยไม่ตั้งใจ
*   **คิดเงินไวใน 5 วินาที (5-Second Checkout):** คำนวณเงินทอนอัตโนมัติเป็นตัวเลขขนาดใหญ่ชัดเจน พร้อมระบบคัดแยกช่องทางรับเงิน (เงินสด, เงินโอน, แอปเดลิเวอรี่) เพื่อการเช็คยอดที่ง่ายดาย
*   **พร้อมโกอินเตอร์ (Global-Ready):** รองรับทั้งภาษาไทยและภาษาอังกฤษ พร้อมระบบปรับเปลี่ยนสัญลักษณ์หน่วยเงินตามทำเลร้านค้าของคุณ

🚀 **CHO-CHA-DA SME Assistant** is a high-performance, intuitive mobile POS and inventory management application built with Flutter. Designed with a "Zero-Learning Curve" and "Visual-First" approach, it empowers busy baristas and small shop owners worldwide to manage sales, track stock levels via swift QR code scanning, and handle checkout logic within 5 seconds.

✨ **Key Core Pillars:**
• 📸 **Scan to Cart:** Accurate multi-item order cart generation with an automated scan-lock buffer.
• 💵 **5-Second Checkout:** Cash-change calculations and payment tagging (Cash, Bank Transfer, Grab).
• 🌐 **Global-Ready:** Single-app codebase with built-in multi-language translation (EN/TH) and dynamic currency support.
• 🛡️ **Enterprise Architecture:** Designed with Material 3 styling and structured to seamlessly scale from a secure local embedded database (Hive/Isar) into a multi-store Cloud SaaS platform powered by Firebase.
